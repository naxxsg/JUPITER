Sistema


Para el Reloj

  • Entrada: Interruptor
  • Sirve como activador simple.
  • Proceso: Arduino Uno, D53231 (RTC)
  • Entrada: Interruptor
  • Sirve como activador simple.
  • Proceso: Arduino Uno, D53231 (RTC)
  • Para procesar la hora en tiempo real.
  • Salida: Pantalla LCD
  • Para mostrar la hora a tiempo real.

Para la Luz

  • Entrada: Interruptor
  • Sirve como activador simple.
  • Proceso: Arduino Uno
  • Para procesar y facilidad de conexión entre los componentes.
  • Salida: RTB (Luxeon):
  • Para generar una luz dentro de la casa.

Para los Cargadores

  • Entrada: Interruptor
  • Sirve como activador simple.
  • Proceso: Batería (1000 mAH) (3,7V)
  • La cantidad de energía que se proporcionará al dispositivo durante una hora.
  • Salida: Puertos USB
  • Cargar dispositivos móviles

Arduino Uno: Procesador

El Arduino cuenta con diferentes conectores que ayudaran a conectar nuestro sistema. Con diferentes salidas que nos permitirán brindar energía en el producto.

Jumpers: Conectores

Es un elemento que permite cerrar el circuito eléctrico del que forma parte dos conexiones.

DS 3231 (RTC): Entrada

Placa de modulo de reloj que permitirá configurar la hora de encendido y apagado del sistema.

Módulo Cargador TP4056: Entrada

Adecua la entrada de energía para el estándar de 1A de la mayoría de baterías de litio. Irá conectada a la batería de la lampara.

Pantalla LCD: Salida

Se utiliza para ver imágenes fijas y en movimiento, en este caso se podrá ver la hora a través de esta pantalla.

Baterías de Litio: Fuente de Energía

La batería será la fuente primordial de energía del producto. Irá conectada al módulo TP4056.

Switch: Entrada

Pequeño elemento que activará el funcionamiento del producto.

RTB Luxeon: Salida

Es una nueva fuente de luz compacta y con un gran rendimiento energético.

I2C: Proceso

Dispositivo unido a la pantalla LCD que permitirá manejar la pantalla de una manera fácil.
